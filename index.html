HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The head element is a container for metadata.
  • The head is placed between the html tag and the body tag.
  • Metadata is not displayed.
  • Metadata is typically defined in the document title, character set, styles, scripts, and and other meta information.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want the content inside an element.
  • Margin - Border - Padding - Content
  • CSS - cascading style sheets
  • DRY code - Do not repeat yourself
  • webpage elements - text, image, header, list

Git

  • git status: check what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • 3 types of GitHub flow are interface, CLI, and desktop.
  • GitHub Pages

JavaScript

  • A variable is a named contained that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Conditional Logic
  • Arrays
  • Loops
  • js function
